# ProjetStage


#Partie FrontEnd:

1)Install Vue.js 

npm install vue

2) Install Vue CLI

npm install -g @vue/cli

3)install vuetify

npm i vuetify

4)install bootstrap-vue

npm i bootstrap-vue

#Partie backEnd:

1) Create an environment:

py -3 -m venv venv

2)Activate the environment

> venv\Scripts\activate

3)Install Flask

python -m pip install Flask 

4)Run the application

 python -m flask run
